# Beca-Sky 🚀

Beca-Sky, Bluesky hesap analizi ve takipçi yönetimi için tasarlanmış Python tabanlı bir araçtır. Platformun kullanıcı arayüzü (UI) rakamları ile gerçek API verileri arasındaki tutarsızlıkları tespit ederek sosyal ekosisteminizi optimize etmenize yardımcı olur.

## 🌟 Özellikler

- Takipçi Analizi: Sizin takip ettiğiniz ama sizi geri takip etmeyen hesapları tespit eder.
- Beyaz Liste (Whitelist) Koruması: Önemli haber kaynaklarını, markaları veya dost hesapları otomatik işlemlerden muaf tutmanızı sağlar.
- Senkronizasyon Boşluğu Tespiti: Silinmiş veya askıya alınmış hesapların (hayalet takipçiler) sayaç üzerindeki etkisini analiz eder.
- Otomatik Yönetim: AT Protokolü kullanarak pasif veya karşılıksız hesapları güvenli bir şekilde takipten çıkarmanızı sağlar.

## 🛠️ Kullanılan Teknolojiler

- Dil: Python
- API Kütüphanesi: [atproto](https://github.com/MarshalX/atproto)
- Geliştirme Ortamı: Visual Studio Code
